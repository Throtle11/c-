# C++ 엔드포인트 파일 스캐너 

Windows 환경에서 폴더를 스캔하여 파일 정보를 수집하고 필터링 후 리포트(CSV/JSON)로 저장하는 C++ 프로젝트입니다.

# 목표
- 엔드포인트 보안 에이전트의 파일 스캔 단계별(정보수집-필터링-리포트저장) 감각 익히기
- RAII(리소스 관리)와 예외 안전성 습관화
- 대량 파일 처리 성능/구조 개선 경험 축적

# 주요 기능 (구현 예정 포함)
-  폴더 재귀 탐색(디렉터리 순회)
-  파일 메타데이터 수집(경로/크기/수정시간)
-  필터링(확장자/크기/기간)
-  결과 출력(콘솔)
-  리포트 저장(CSV)
-  (선택) JSON 저장
-  멀티스레드 스캔(워커 풀)

# 개발 환경
- OS: Windows
- IDE: Visual Studio 2022
- Language: C++
